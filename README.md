# 📦 Sistema de Controle de Estoque

Projeto simples desenvolvido em Python usando Flask para controle de estoque.

---

## 🚀 Funcionalidades

* ➕ Adicionar produtos
* ➖ Remover produtos por quantidade
* 🔄 Repor produtos
* 💰 Cálculo automático do valor total do estoque

---

## 🛠️ Tecnologias utilizadas

* Python
* Flask
* HTML
* CSS

---

## ▶️ Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/seuusuario/sistema-estoque.git
```

2. Acesse a pasta do projeto:

```
cd sistema-estoque
```

3. Instale as dependências:

```
pip install flask
```

4. Execute o projeto:

```
python appy.py
```

5. Acesse no navegador:

```
http://127.0.0.1:5000
```
